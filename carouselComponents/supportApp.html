Support App

Tools

C# (Xamarin), SMSS, Jira (API), DevOps (Azure)

Description

This was my from scratch made Support App. During my 3rd year of my study I had an internship at CaptureTech. Here I started the development of the Support App. After logging in users get access to many options.

Using an API to communicate with the Jira service an user can:
  • Send a problem message including a picture or video if needed,
  • Call the Service Desk,
  • Use a FAQ to solve easy and common problems,
  • Receive notifications when a problem is already send by a different user,
  • Receive notifications when a problem is picked up by the Service Desk with an indication on how long it will take.
